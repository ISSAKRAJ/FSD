

# Digital Board

A modern, interactive digital board application designed for collaboration, brainstorming, and project management. It allows users to create, organize, and share ideas visually in real-time.


## 📌 Features
- **Interactive Canvas**: Drag, drop, and resize notes, tasks, or widgets.
- **Real-Time Collaboration**: Multiple users can edit simultaneously.
- **Customizable Boards**: Create boards for projects, meetings, or personal planning.
- **Persistence**: Save and load boards with cloud or local storage.
- **Cross-Platform**: Works on desktop and mobile devices.



## 🚀 Getting Started

### Prerequisites
- Node.js (>= 18.x)
- npm or yarn
- Modern browser (Chrome, Edge, Firefox)

### Installation
```bash
git clone https://github.com/yourusername/digital-board.git
cd digital-board
npm install
```

### Run Development Server
```bash
npm run dev
```
Open `http://localhost:3000` [(localhost in Bing)](https://www.bing.com/search?q="http%3A%2F%2Flocalhost%3A3000%2F") in your browser.

---

## 🛠️ Usage
1. Create a new board.
2. Add sticky notes, tasks, or shapes.
3. Invite collaborators via share link.
4. Export board as image or PDF.
